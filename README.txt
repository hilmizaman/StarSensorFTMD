Cara set up program:
1. Install Windows Subsystem for Linux.
2. Buat user WSL kemudian install anaconda.
3. Clone repository menuju folder di WSL.
4. Install library yang digunakan dalam program

Cara menjalankan program:
1. Terdapat 2 program utama untuk melakukan star centroiding dan star identification
2. Pastikan program menggunakan katalog dan sumber gambar yang benar
3. Jalankan program star centroiding dan output hasil program dalam format .txt.
4. Jalankan program star sensor menggunakan metode yang diinginkan.
5. Output attitude yang didapat menuju bentuk attitude lain.
